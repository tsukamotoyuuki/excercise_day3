
ex1.
numbers = [12, 45, 7, 89, 23, 67] があります。
このリストの中から最も大きい数値を探し出して出力してください。


ex2.
文字列 password = "secret1234" があります。
最初の2文字と最後の2文字だけをそのまま残し、間の文字をすべて * に置き換えた文字列を
作成して出力してください。（出力例: se******34）

ex3.
変数 n = 5 があります。1から n までの全ての整数を掛け合わせた結果
（5の階乗 = 5 * 4 * 3 * 2 * 1）を、計算し出力してください。


ex4.
リスト logs = ["INFO: boot", "ERROR: db connection", "WARN: high load", "ERROR: timeout"] があります。"ERROR" で始まる要素がいくつあるかをカウントして出力してください。

ex5.
リスト scores = [55, 92, 38, 92, 75, 81] があります。
このリストの中で「2番目に大きい数値」を見つけて出力してください。
なお、同じ最大値が複数ある場合（この例では92）は、次に大きい数値（81）を2番目とみなします。


ex6.
リスト items = [0, 1, 0, 3, 12] があります。
0以外の数字の相対的な順序は保ったまま、すべての 0 をリストの末尾に移動させた新しいリストを
作成して出力してください。（出力例: [1, 3, 12, 0, 0]）


ex7.
リスト instances = ["db-server", "web-server-01", "cache", "load-balancer"] の中で、文字数が最も多い文字列を探して出力してください。

ex8.
1~6までのランダムな数を出力し（6面体のサイコロ）、「1」が出るまで続けてください。
また「1」が出るまでに合計何回サイコロを振ったかも出力してください。

ex9.
リスト errors = ["404", "500", "404", "403", "500", "500"] があります。
各エラーコードが何回出現したかを集計して出力してください。
（出力例：{"404": 2, "500": 3, "403": 1}

ex10.
文字列 ip = "192.168.1.999" があります。
ドットで区切られた4つの部分がすべて 0 から 255 の範囲の数値であるかを判定し、
正しいIPv4アドレスなら True、不正なら False を出力してください。
